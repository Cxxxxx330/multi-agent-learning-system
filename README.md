# 🧠 Multi-Agent Learning System

A multi-agent AI system that automates study planning, tutoring, evaluation, and personalized optimization.

---

## 🚀 Features

- 📅 Automatic learning plan generation
- 🤖 Multi-agent collaboration (Planner / Tutor / Evaluator / Memory)
- 🧠 Context-aware memory system
- 📊 Learning performance evaluation
- 🔁 Feedback loop optimization

---

## 🏗 Architecture

User Input  
↓  
Planner Agent → Task Decomposition  
↓  
Tutor Agent → Explanation  
↓  
Evaluator Agent → Testing  
↓  
Memory Agent → Store & Optimize  

---

## 🛠 Tech Stack

- Python
- LLM API (OpenAI / Claude)
- Vector Database (FAISS, optional)
- Multi-Agent Architecture
